# ngx-admin

# Admin template based on Angular

## Repository state

Repository is currently in a state of minimal maintenance. The primary focus is on keeping the Angular version used in this project up to date.

## Installation notes

To install ngx-admin you have to use NodeJS version 14.14+ because of [node-sass](https://github.com/sass/node-sass) version utilized in the application.

## Key features

- Angular open source dashboard template.
- Over 40+ Angular Components and 60+ Usage Examples.
- Material theme is backward-compatible.

### What's included:

- Angular & Typescript
- Bootstrap 4+ & SCSS
- Responsive layout
- RTL support
- High resolution
- Flexibly configurable themes with **hot-reload** (3 themes included)
- Authentication module with multiple providers
- 40+ Angular Components
- 60+ Usage Examples

## Material theme for ngx-admin

To use material theme checkout `feat/material-theme` branch.
